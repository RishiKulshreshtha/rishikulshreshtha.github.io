# rishikulshreshtha.github.io

> Personal portfolio with a twist — unlock sections by winning mini-games.

**[rishikulshreshtha.github.io](https://rishikulshreshtha.github.io/)**

---

## Stack

```
Frontend    Svelte 5 · Vite 6 · CSS Custom Properties
Audio       Web Audio API (synthesized — zero external files)
Icons       Font Awesome 6 (self-hosted)
Fonts       Press Start 2P · VT323 · Inter
Testing     Vitest · Playwright
Hosting     GitHub Pages · GitHub Actions CI/CD
```

---

## Run locally

```bash
npm install
npm run dev
```

```bash
npm test             # unit tests
npm run test:e2e     # E2E tests
npm run build        # compile → docs/
```
